# Agent Hive v2.4 RFC PR-Ready Bundle

This directory is laid out as a repo overlay.

To apply it manually to `intertwine/hive-orchestrator`:

1. copy `docs/V2_4_STATUS.md` into `docs/`
2. copy `docs/hive-v2.4-rfc/` into `docs/`
3. update `pyproject.toml` using `PACKAGE_INCLUDE_NOTES.md`
4. use `PULL_REQUEST_BODY.md` as the starting PR description

Key implementation docs live under `docs/hive-v2.4-rfc/`.
