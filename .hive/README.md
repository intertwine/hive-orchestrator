# Hive 2.0 Substrate

This directory contains the canonical machine-readable state for Hive 2.0.

- `.hive/tasks/*.md` stores canonical task records.
- `.hive/runs/*` stores immutable run artifacts.
- `.hive/memory/*` stores project-local memory docs.
- `.hive/events/*.jsonl` stores append-only audit logs.
- `.hive/cache/` is derived and gitignored.
- `.hive/worktrees/` is local scratch space and gitignored.
